This plugin extends the WooCommerce e-commerce plugin by allowing a custom product tab to be created with arbitrary content.

== Description ==

Adding multiple custom tabs in woocommerce product details page.

The lumia product tab, if defined, will appear in between the 'additional information' and 'reviews' tabs.

To easily add multiple tabs, share tabs between products, and more features.

== Installation ==

1. Upload the entire 'lumia-product-tabs' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit a product, then click on 'Product Tabs' within the 'Product Data' panel
